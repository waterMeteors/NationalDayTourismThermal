## 一、项目介绍
本项目是通过爬虫计算获取热点景点和相关旅游类型进行数据获取准备，同时通过pyecharts
将数据进行分析处理
## 二、依赖包说明
 * python3：python的版本
 * csv
 * pyecharts
 * fake_useragent
 * requests
 * bs4 </br>
 
 注意:项目clone后请导入相关依赖
## 三、项目的大致结果
    可以查看项目中的render.html文件。由于爬虫是否耗时间，故将数据爬取与数据分析
    写道两个python文件中
